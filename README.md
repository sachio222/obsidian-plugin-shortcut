# Show Plugins Shortcut in Sidebar

Adds a puzzle piece icon to the left ribbon that opens the installed community plugins screen in one click.

## Usage

Click the puzzle piece icon in the left sidebar. Settings opens directly to the Community Plugins tab.

## Why?

Getting to your installed plugins normally takes: Settings > Community Plugins. This saves a click and puts it where you can see it.

## Installation

Copy the plugin folder to `.obsidian/plugins/plugin-store-shortcut/` and enable it in Settings > Community Plugins.
